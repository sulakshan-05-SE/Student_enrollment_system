Student Course Enrollment System

This is a web-based Student Course Enrollment System developed using HTML, CSS, Bootstrap, JavaScript, and PHP.
The system provides complete management for students, teachers, courses, and enrollments.

Features
Student management
Teacher management
Course management
Enrollment management
Add, edit, and delete records
Enroll a student into multiple courses
View all enrollment details
Generate different types of reports
Responsive user interface using Bootstrap
Server-side validation
MySQL database integration

Technologies Used
HTML
CSS
Bootstrap
JavaScript
PHP
MySQL
